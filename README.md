# 🤖 Coze-AI职场办公效率专属助手
基于字节跳动【扣子Coze】平台搭建的零代码AI助手，专注职场办公场景，提供标准化周报模板、职场沟通话术、工作效率规划等落地能力，输出干净无冗余，可直接复制使用。

## 🎯 项目简介
本项目通过 **工作流编排 + 私有知识库 + 强制Prompt规则** 实现定制化AI助手，解决职场办公高频需求，全程屏蔽模型思考过程，纯成品输出。

## ✨ 核心功能
- 📝 多场景周报自动生成（通用/技术/结果导向/Prompt工程师专属）
- 🗣️ 职场沟通话术、工作汇报方案输出
- 📊 工作规划、效率提升建议
- 🛡️ 严格知识库调用，拒绝虚假信息
- 🧹 100%干净输出，无思考过程、无冗余废话

## 🛠️ 技术栈
- 开发平台：字节扣子 Coze（零代码AI编排）
- 大模型：GLM-4.7
- 核心能力：知识库检索、对话工作流、全局规则约束
- 部署方式：扣子平台一键发布

## 🚀 快速搭建教程
### 1. 注册扣子平台
打开 https://www.coze.cn/ ，手机号注册登录

### 2. 创建AI助手
1. 点击「创建助手」→ 命名：职场办公效率助手
2. 配置人设与全局规则（见下文核心配置）

### 3. 搭建对话工作流
1. 添加「意图选择器」：写作、沟通、效率、兜底
2. 绑定「知识库检索」节点
3. 添加「大模型文本生成」节点
4. 配置「结束节点」统一输出

### 4. 上传知识库
添加职场文档、周报模板、沟通话术等私有数据

### 5. 一键发布
配置完成后，点击右上角「发布」即可使用

## ⚙️ 核心配置
### 全局人设强制铁则（最高优先级，凌驾所有默认逻辑）
1. **身份语气**：资深职场效率专属私人顾问，全程专业友好、干练得体，表达简洁精准，拒绝冗长啰嗦
2. **内容来源**：优先严格调用专属职场私有知识库内容，绝不编造虚假信息，方案全部可直接照搬套用
3. **屏蔽约束**：100%彻底隐藏模型思考、推理草稿、溯源过程等全部冗余内容，仅交付干净最终成品
4. **排版输出**：全程结构化，使用标题+分段+列表展示，工整清爽，支持一键复制
5. **兜底原则**：知识库无匹配内容时，礼貌给出合理通用建议，绝不凭空捏造不实信息
6. **上下文承接**：完整记住对话历史，连贯承接多轮追问，逻辑全程统一
7. **净化规则**：严禁「为你整理如下」这类多余开场白/过渡话术，开门见山，直接输出正文
