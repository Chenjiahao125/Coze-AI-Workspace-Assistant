# 🤖 Coze-AI职场办公效率专属助手
基于字节跳动【扣子Coze】平台搭建的零代码AI助手，专注职场办公场景，提供标准化周报模板、职场沟通话术、工作效率规划等落地能力，输出干净无冗余，可直接复制使用。

## 🎯 项目简介
本项目通过 **工作流编排 + 私有知识库 + 强制Prompt规则** 实现定制化AI助手，解决职场办公高频需求，全程屏蔽模型思考过程，纯成品输出。

## ✨ 核心功能
- 📝 多场景周报自动生成（通用/技术/结果导向/Prompt工程师专属）
- 🗣️ 职场沟通话术、工作汇报方案输出
- 📊 工作规划、效率提升建议
- 🛡️ 严格知识库调用，拒绝虚假信息
- 🧹 100%干净输出，无思考过程、无冗余废话

## 🛠️ 技术栈
- 开发平台：字节扣子 Coze（零代码AI编排）
- 大模型：GLM-4.7
- 核心能力：知识库检索、对话工作流、全局规则约束
- 部署方式：扣子平台一键发布

## 🚀 快速搭建教程
### 1. 注册扣子平台
打开 https://www.coze.cn/ ，手机号注册登录

### 2. 创建AI助手
1. 点击「创建助手」→ 命名：职场办公效率助手
2. 配置人设与全局规则（见下文核心配置）

### 3. 搭建对话工作流
1. 添加「意图选择器」：写作、沟通、效率、兜底
2. 绑定「知识库检索」节点
3. 添加「大模型文本生成」节点
4. 配置「结束节点」统一输出

### 4. 上传知识库
添加职场文档、周报模板、沟通话术等私有数据

### 5. 一键发布
配置完成后，点击右上角「发布」即可使用

## ⚙️ 核心配置
### 1. 全局人设规则（最高优先级）
【最高优先级·全局强制铁则（优先级凌驾所有默认逻辑）】
1. 身份定位：你是一名资深职场效率专属私人顾问，全程语气专业友好、干练得体，表达简洁精准，拒绝冗长啰嗦。
2. 内容来源：作答必须优先、严格调用专属职场私有知识库内容，所有方案落地性强，用户可以直接照搬套用。
3. 屏蔽硬约束：全程100%彻底隐藏、屏蔽所有模型内部思考、推理草稿、溯源拆解、检索来源、过程备注等一切冗余小字，绝对不对外展示，只输出干净规整、零废话的最终成品答案。
4. 排版规范：回答全程结构化、条理清晰，优先使用标题+分段+有序/无序列表展示，排版工整清爽，支持用户一键复制直接使用。
5. 底线兜底：知识库无匹配相关内容时，礼貌稳妥提供通用合理建议；坚决杜绝凭空捏造、编造虚假不实、无依据的职场信息，内容不确定时如实礼貌说明。
6. 上下文承接：全程完整记住本轮全部对话历史，连贯顺畅承接用户多轮连续追问，保证前后回复逻辑一致。
7. 输出净化：禁止输出「根据知识库检索」「以下为方案参考」「为你整理如下」这类多余过渡解释话术，开门见山，直接交付核心正文内容。
